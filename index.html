PROJECTS



Community Fridge Site

The concept of this website is to allow community fridge users to have digital access to their fridges.

Interactions include: adding and checking out food, add food to a fridge, editing a fridge, and adding a new fridge.

Over the course of four months, the Community Fridge Website gained more of these features. I started off coding using just HTML, CSS, and javascript. Using Node.js, I added a server to process user requests. Later on I used Express to create a REST API and tested the routes using Postman before testing the site.

Server Client Library System

This project was done for my Intro to System Programming course, where we used C.

The library program contains to processes that communicate using TCP/IP sockets.

The server side is handling all the data from the library & handles requests sent from the client. The client side allows a user to make a request from the menu - either viewing the books in the library or checking in/out a book. I thought it was quite fun to see both termials at work!







Sheep Run

In May 2020, I created Sheep Run using Python and Pygame. The concept is based off The Dinosaur Game by Google but I designed my own images using Adobe Illustrator. I had a lot of fun coding the different moving elements, espically the sheep jump!

In the future, I hope to rewrite the game in HTML - to make it more accessible to play.
